gunit instructions
------------------

If a change is made to Pass.g
-----------------------------
java org.antlr.Tool Pass.g // re-set main grammar file if Pass.g is modified
javac *.java // compile all files
java org.antlr.gunit.Interp AtomTests.gunit // run AtomTests.gunit tests


Testing with unmodified Pass.g
------------------------------
java org.antlr.gunit.Interp AtomTests.gunit // run AtomTests.gunit tests
